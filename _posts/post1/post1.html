Dataset

This coronavirus dataset has the following fields as seen in Table 1. The dataset is obtained from coronavirus package in CRAN. Krispin and Byrnes (2020)

Table 1: Coronavirus dataset variables
variable class description
date Date The date of the summary
country character The province or state, when applicable

cases integer the number of daily cases (corresponding to the case type

To make comparisons, the dataset from tidycovid19 is also used in this report. Gassen (2020) This dataset has more variables because it is a combination of several other dataset sources. The variable from tidycovid19 listed below in Table 2. For this analysis, we only use the confirmed, deaths and recovered which contain cumulative number of cases.

Table 2: tidycovid19 dataset variables
variable class description
confirmed numeric Confirmed Covid-19 cases as reported by JHU CSSE (accumulated)
deaths numeric Covid-19-related deaths as reported by JHU CSSE (accumulated)
recovered numeric Covid-19 recoveries as reported by JHU CSSE (accumulated)

Daily Cases

Based on a report from Anadolu Agency Karaalp (2020) the first case in Iraq occurred on 24 February. After that the second case was reported two days later with a jump of 4 cases at once. Entering March, daily cases tended to fluctuate. However, the cumulative cases increased significantly. The first case of death due to Covid-19 was reported on March 4, as many as 2 cases. Until the last data is obtained on the date 2020-08-31 a total of 234934 positive cases, 7042 died, and 176602 declared cured.

To see the development of daily cases in Iraq, we can check the data from Table 3 below. It contains the data of daily cases from 22 January 2020 until 31 August 2020. Alongside the table, a graph is provided as well in Figure 1.
